Python requirements: pip install glob2 termcolor psutil

On Windows you can run tests with: fisd2/test/run_tests.bat

On Linux you can run tests with : fisd2/test/run_tests.sh

Results are stored in: fisd2/test/reports 
	Simple report: fisd2/test/reports/report.txt 
	Testing log  : fisd2/test/reports/report.txt.log 
	Testing stat : fisd2/test/reports/report.txt.json
